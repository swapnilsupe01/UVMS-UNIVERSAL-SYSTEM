Capabilities

Everything in one scan

🎯

Object Detection

Identifies 600+ object classes including cars, people, furniture, animals and more using YOLOv8-Large with confidence scores and bounding boxes.

🌊

Depth Estimation

Generates a per-pixel depth map using Intel MiDaS — visualized as a colorized heatmap showing near and far regions of any scene.

📐

Real-World Measurements

Estimates actual width, height, and ground clearance of detected objects by combining bounding box size with MiDaS depth values.

🚗

Vehicle Analysis

Detects cars, trucks, buses and motorcycles. Estimates ground clearance in cm using depth data — useful for road and parking analysis.

🛣️

Road Damage Detection

Detects potholes, alligator cracks, longitudinal and transverse cracks. Rates severity as Critical, Urgent, Moderate or Minor with repair priority.

🦁

Wildlife Detection

Identifies animals and birds across 20+ species. Groups detections into pets, farm animals, wild animals and birds for easy classification.

🏢

Building Detection

Detects buildings, skyscrapers, bridges, churches and more using TensorFlow SSD trained on Open Images. Estimates floor count from height.

🏠

Interior & Furniture

Recognizes chairs, couches, beds, tables, wardrobes and more — useful for home, office and building interior scans and inventory.

⚡

GPU Accelerated

Runs on Tesla T4 GPU via Google Colab — fast inference across all 9 endpoints, exposed publicly via Cloudflare tunnel.
